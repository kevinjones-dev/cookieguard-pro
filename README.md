# CookieGuard Pro

A modern, privacy-first cookie manager for Chrome — edit, manage, and auto-delete cookies with zero data collection.

## Features

- View all cookies for any website
- Edit any cookie field — name, value, domain, path, expiration, secure, httpOnly, sameSite
- Delete individual cookies or all cookies for a site
- Auto-delete cookies when you close a tab
- Whitelist sites to protect their cookies from auto-delete
- Import and export cookies as JSON
- Search and filter cookies by name
- Dark and light themes
- Clean, professional UI

## Privacy

This extension collects zero data. No analytics, no tracking, no telemetry. Nothing ever leaves your device. All settings are stored locally and deleted automatically when you uninstall.

Read the full [privacy policy](https://kevinjones-dev.github.io/cookieguard-pro/privacy-policy.html).
