# CookieGuard Pro

A modern, privacy-first cookie manager for Chrome — the replacement for EditThisCookie and Cookie AutoDelete.

**Status:** Phase 1 (MVP) — View and delete cookies.

## Features (Phase 1)
- View all cookies for the current website
- Delete individual cookies
- Delete all cookies for the current site
- Clean dark-theme UI
- Zero data collection — everything stays on your device

## Roadmap
- Phase 2: Cookie editing (name, value, expiration, flags)
- Phase 3: Auto-delete on tab close + whitelist
- Phase 4: Import/export cookies as JSON
- Phase 5: Polish, light theme, professional icons
- Phase 6: Chrome Web Store launch
- Phase 8: Premium features via ExtensionPay

## Privacy
This extension collects zero data. Nothing ever leaves your device.
All cookie data stays in your browser. Read the full privacy policy in `privacy-policy.html`.
